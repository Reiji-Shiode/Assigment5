## URL
- **Uniform Resource Locator**の略で、インターネット上のホームページ（Webサイト）やファイルの位置や情報を示すもの

![URL](https://ferret.akamaized.net/images/59e815a4fafbd876ce0003e4/original.jpg?1508382115)


## クエリ文字列（クエリパラメーター）
- サーバーに情報を送るためにURLの末尾に付け足す文字列（変数）のこと  
- `「？」`を末尾に付け、その次に`「パラメーター＝値」`を付ける  
- 複数のパラメーターを付けたい場合は`「＆」`を使用する  

![クエリ文字列](https://web-knowledge-info.com/wp/wp-content/uploads/2020/07/%E3%82%AF%E3%82%A8%E3%83%AA%E6%96%87%E5%AD%97%E5%88%97.jpg)

## パス変数（パスパラメーター）
- ディレクトリやファイルへのショートカット

## クエリ文字列とパス変数の違い
〇見た目の違い  
> 1 https://zenn.dev/search  
> 2 https://zenn.deb/search?=Laravel  
> 
> 1のパス変数は「search」の部分、2の場合、パス変数は1と同様に「search」、クエリ文字列は「？q＝Laravel」になる

- 例として　株式会社アニメ（ドメイン：Anime.vo.jp）に営業部（Sales）があり、チームが以下のように分かれているとする

〇SalesTable
id|name
-|-
1|Isono
2|Doraemon

〇UsersTable
id|saled_id|name
-|-|-
1|1|サザエ
2|1|カツオ

4|2|のび太
5|2|ドラえもん
6|1|波平

 営業部Isonoをチームのページを表示するとなるとURLは以下のようになる  
> https://Anime.co.jp/sales/{group_id}  

- パス変数は`特定のもの（画面など）を表示したいとき`に必要になる  
> 例: https://Anime.co.jp/sales/1  

- クエリ文字列は`特定のもの（画面など）に条件を加える場合`に必要になる。  
> 例: 上記メンバー一覧から特定の人を検索したい場合（今回の場合はID検索と想定）  
> 今回はUserTableのID:3（ワカメ）を検索（条件の追加）  
> https://Anime.co.kp/sales/1/members?id=3  

## HTTPメソッド
- **HTTPメソッド**はクライアントがサーバーにどんなことをしてほしいのか教えるときに使う機能  

名称|意味
-|-
GET|リソース情報を取得する
POST|新しいリソース情報を送り込む
PUT|リソース情報を新しい情報で置き換える
PATCH|リソース情報の一部を新しい情報で置き換える
DELETE|リソース情報を削除する

## リクエストヘッダー
- リクエストは3つの部品から構成されており、**リクエストヘッダー**はその3つの部品のうちの1つとなっている

## HTTPSステータスコード
- HTTPレスポンスに含まれるWebサーバーの処理結果を表現する3桁の数字のこと  

2××成功|意味
-|-
200|リクエスト成功
201|リクエストが成功してリソースの作成が完成

4××クライアントエラー|意味
-|-
400|リクエストが無効
404|リソース、ページが存在しない

5××サーバーエラー|意味
-|-
500|サーバー内部エラー

## JSON
- JavaScriptのオブジェクト記法を用いたデータ交換フォーマット  
 - JSONを挟むことで各プログラミング言語間のデータの受け渡しが容易になる  

 〇JSONの記載方法
> ・{}（波括弧）で囲む  
> ・それぞれのデータは、キー（key）と値（value）のペアにする必要がある  
> ・keyとvalueは:（コロン）で区切る  
> ・それぞれのデータを,（カンマ）で区切る（最後は不要）  
> ・keyは文字列、valueは様々なデータ型で記述可能  
> ・文字列は必ず""（ダブルクオーテーション）で囲む  

## JSONを用いたデータの表現
```
{
    "id":1,
    "name":"Shiode",
    "gender":"male",
    "birth":"1999/06/06",
    "hobby":"Cafe-hopping"
}
```

     System.out.println("バッククオートを上下に3つずつ記載することでコードも書ける！”);
